Design of Distributed Key-Value Store With Raft Consensus Algorithm

◦ Implemented Key-Value store in Go language and used this in a cluster setup.

◦ Used ZMQv4 library for communication purpose at socket level.

◦ Leader election and log replication among the nodes in the cluster was done using Raft consensus algorithm
